#### 项目简介
    提供一套以组件化开发的脚手架，能够快速进入开发


#### 项目背景

    基于Jetpack组件，并且使用Kotlin语言进行开发  
    目的是为了快速进行开发，拉下代码基于本项目就能直接开发。  
    本项目使用了大量的Kotlin特性和语法，简化了大量的代码。
    并且封装了一些常用的的功能。能够开箱即用，快速开发
